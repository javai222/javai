<h3><b><i>Project Name:</i></b><i> Bulldogs MarketHub</i></h3>

<h4><i>📃 Description:</i>
Students and staff at National University Laguna can purchase and sell products on campus using the NU Marketplace/Food Grab App. 
Users can choose to act as buyers or sellers, manage their accounts, and register using their NU email for verification (Students/Staff).

<br>👤 Administrators may see records and create reports, and users can add, search, and manage products for sale. 
The platform aims to organize the campus marketplace in addition to providing a secure and useful marketplace for devices, accessories, food, and clothing.</br>

<i>Target Users:</i>
Students
Employees
</h4>

<h3><b><i>Simple Use Case Statements:</i></b></h3>



<br><i>1.</i> Users can create an account using their NU email or a personal/business email along with their personal details.
   If an NU email is used, the system verifies whether the user is a student or employee. If a personal or business
   email is used, the user is automatically registered as an S-Lane vendor and will be required to submit additional 
   verification details.</br>
<br><i>2.</i> The user will log-in their username and password so that they can enter the app.</br>
<br><i>3.</i> The user selects whether they want to buy or sell items. If the user selects the buyer option, the user’s account
   is limited to only buying items on the marketplace. If the user selects the seller option, they will have to provide
   additional information and need to be verified, they will also have access to buy other items from the marketplace.</br>
<br><i>4.</i> The user can view their own personal records and other public records. The admin can view all records.</br>
<br><i>5.</i> The system allows users to modify their personal details. Business vendors may also update their information such as
   business status, when closing or modifying their shop.</br>
<br><i>6.</i> The user can delete their personal records. The Admin can delete any existing records.</br>
<br><i>7.</i> The user can list or add items or products to be sold to other users in the platform.</br>
<br><i>8.</i> The user may delete the item they posted if it has been already sold or if they decided to change their mind.</br>
<br><i>9.</i> The user can search items available. The user can enter a keyword to find a specific item. The user can view the list
   of items that match the search. The user can clear the search to view all of the items again.</br>
<br><i>10.</i> The user can generate their own transaction records. The admins can generate system reports, transactions,
    and other activities.</br>
<br><i>11.</i> The account will be deleted if the user deleted it themselves, inactive for a long time, the user has violated rules,
    and did not verify their account according to their due date.</br>
<br><i>12.</i> The chat support system is a digital platform that enables consumers to communicate with support agents in real time
    and receive prompt assistance.</br>
<br><i>13.</i> The system can verify students and employees once every year. The system can verify S-lane vendors once every month.</br>
